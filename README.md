# eRepairing.com Frontend

React/Next.js frontend for eRepairing.com Service Management Platform.

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Multi-role dashboards (Customer, Engineer, City/State/Country/Org Admin, Platform Admin, Vendor)
- Real-time ticket management
- AI-powered features integration
- Inventory management
- Warranty tracking
- Analytics and reporting
